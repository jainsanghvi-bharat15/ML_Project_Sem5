# Olist Brazilian E-commerce Dataset

## About the Dataset
The dataset being analyzed here is the Olist Brazilian e-commerce dataset. This dataset contains data about orders, sellers, products, payment methods, reviews, and customers' locations in Brazil.

## Directory Structure
- `data/` -> raw and processed csv files
- `notebooks/` -> Jupyter Notebooks
- `models/` -> Machine Learning models
- `figures/` -> Plots and dashboards created
- `src/` -> Python scripts used for cleaning the data and feature engineering
- `reports/` -> Summaries and plots created from the analysis
